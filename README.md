# 🏥 HealthGuardian AI

An AI-powered rural healthcare supervision platform built for the Google Agentic AI Competition.

## 🚀 Features

- Google Authentication
- Firestore Database
- Attendance Check-in
- GPS Verification (Coming Soon)
- Multi-Agent AI Architecture
- Health Policy RAG
- Fraud Detection
- Analytics Dashboard

## 🛠 Tech Stack

### Frontend
- React
- Vite
- Firebase

### Backend
- FastAPI
- Python

### AI
- Gemini
- LangGraph (planned)
- FAISS (planned)
